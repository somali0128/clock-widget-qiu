梦幻星空小狐狸时钟挂件 使用说明

一、这个文件夹是什么？

这是一个可以放进直播画面的时钟挂件。

你真正需要导入直播软件的是：

index.html

推荐尺寸：

900 x 600


二、导入直播姬的方法

1. 打开直播姬。
2. 进入你的直播场景。
3. 找到“添加素材”或“添加来源”。
4. 选择“浏览器源”“网页源”或类似名字的选项。
5. 如果可以选择本地文件，就选择本文件夹里的 index.html。
6. 宽度填写 900。
7. 高度填写 600。
8. 确认添加。
9. 在画面里拖动时钟，放到你喜欢的位置。

如果画面太大：

直接在直播姬里缩小这个素材即可。

如果出现黑底或白底：

检查浏览器源设置里有没有“背景色”“背景填充”之类的选项，尽量关掉或设为透明。


三、导入 VTube Studio 的方法

VTube Studio 本身主要是虚拟形象软件，不同版本和插件的网页导入方式可能不一样。

如果你的 VTube Studio 或相关插件支持网页 / 浏览器源：

1. 找到添加网页、Web Item、Browser Source 或类似功能。
2. 选择本地文件。
3. 选择本文件夹里的 index.html。
4. 尺寸设置为 900 x 600。
5. 把时钟拖到合适的位置。

如果你的 VTube Studio 不能直接添加网页：

可以先把这个时钟添加到 OBS 或直播姬里，再一起推流。


四、怎么改时间显示？

打开这个文件：

setting.json

你会看到类似这样的内容：

{
  "time": {
    "showHour": true,
    "showMinute": true,
    "showSecond": false,
    "separator": ":"
  },
  "date": {
    "showDate": true,
    "showYear": false,
    "showMonth": true,
    "showDay": true,
    "showWeekday": true,
    "weekdayStyle": "short",
    "separator": "，"
  }
}

true 代表显示。
false 代表不显示。

想显示秒：

"showSecond": true

不想显示日期：

"showDate": false

不想显示周几：

"showWeekday": false

想显示“星期一”而不是“周一”：

"weekdayStyle": "long"

改完后保存 setting.json，然后刷新直播软件里的浏览器源。


五、常见问题

1. 改了 setting.json 没变化怎么办？

先刷新浏览器源。
如果还是没变化，删除浏览器源后重新添加一次 index.html。

2. 时钟太大怎么办？

在直播软件里缩小这个浏览器源。

3. 时钟周围有白色背景怎么办？

这是截图或预览软件可能显示出来的颜色。
实际放在直播软件里时，页面背景是透明的。
如果直播里也有底色，请检查浏览器源背景设置。
